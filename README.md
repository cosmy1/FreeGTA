# FreeGTA

FreeGTA is an open source reimplementation of the classic **Grand Theft Auto** created through reverse engineering techniques.

## Status
FreeGTA is currently in early development.

## Original Game Compatibility
This reimplementation is based on analysis of the **Rockstar Games Free Release (2004)**

Grand Theft Auto.exe SHA256:
6A5C313D9C0D87B4EFA527BAAC7097DD4A149B9B2DA5261ABB224083F5CDC1C2

## Installation
FreeGTA does **not** include copyrighted game assets.
Users must provide their own legally obtained copy of the original game data and place them inside:
```
Bin\GTA
```

Compile and place `dplayx.dll` file into the following directory before running the game:

```
Bin\GTA\WINO
```
